Overview

Three environment pipeline that deploys a Flask app to Azure App Service using Bicep and GitHub Actions. Dev and QA deploy automatically and Production requires approval, followed by a safe slot swap. Application Insights captures telemetry and a health endpoint supports smoke tests.

Key outcomes

  • IaC with Bicep
  • CI tests and CodeQL
  • CD with approvals and slot swap
